# US Small Business Administration Open Data (data-sba-gov)

US Small Business Administration Open Data is a federal government open-data portal for United States running **CKAN**, exposing a standardized catalog API over ~41 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-sba-gov/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-sba-gov-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.2

## API
- **US Small Business Administration Open Data CKAN Action API** — base `https://data.sba.gov/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.sba.gov)
- Live endpoint verified: `https://data.sba.gov/api/3/action/status_show` · ~41 datasets
